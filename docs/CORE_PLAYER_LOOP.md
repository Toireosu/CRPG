* Design Character

* Exploration [Loop]
    * Player move across the world
    * __ Encounter? --> [Combat/Encounter Resolution] --> back to Exploration
    * __ NPC? --> [Dialogue/Quest/Trade/Combat] --> back to Exploration
    * __ Object? --> [Interaction/Character skill] --> back to Exploration
    * Any of these could be considered a quest objective --> Quest updated

* Combat
    * Turn based
    * Player does action
    * Player NPC does action
    * Enemies does action
    * Continue until --> [Flee/Team death]

* Check player death or finished game
* Game Over / Load from save / Back to Menu

Sum: Player progresses by gaining experience, quests give a lot of experience.
