# Core Systems

## Input system
* Mouse manager
    * Should be clearly design to know precendences and be able to pause different parts of the game based on what is happening.  

## Exploration
* Fog of war. 
* Map renderer & builder
    * Multiple layers far background, floor, entity layer, roofs. 
* Camera scroll
* Exit grids leading to other parts of the world.
* Navigation.

## Character
* Stats.
    * Should be fun and meaningful.
* Inventory
    * Easy modern look.
    * Items. 
* Parameter based look. (he/she/they) (hair-color) (armor)
* Animations

## UI
* Dialogue.
* Inventory.
* Character sheet and level up.
* Main Menu.
* Pause Menu. 

## NPC
* Should have personality. 
* Fighting.
* Dialogue with trees and vars.
* Give quests, notes and finish quests.

## Combat
* Turn controller
* Enemy AI
* Player UI
* Attack -> React
* Chance based yet strategic. 
* Action points. 

## Audio system 
* Should be able to keep playing music even when pausing. 

## Random system
* Should be deterministic.
